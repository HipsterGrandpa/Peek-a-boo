Media Queries

Media queries allow us to make our sites responsive We can use media queries to apply CSS styling only wien certain conditinos apply. For instance, our sites can look different depending on the size of the screen, or viewport a user is viewing our content with. We can also change the way or site appears if a user is print our website, or using a screen-reading!

Using media queries and responsive design allows us to ensure that our site looks and works great no matter how the user is viewing it.

And, as more and more people use more and more devices to browse the internet, integrating media queries into websites is becoming a common, widespread practice!
